# FarmConnect Project Report

## Project Title

FarmConnect: Direct Farmer-to-Consumer Marketplace

## Problem Statement

Farmers often receive low profits due to intermediaries.
Consumers pay higher prices while farmers earn less.

## Proposed Solution

A digital platform connecting farmers directly with consumers.

## Features

- User Authentication
- Product Management
- Order Management
- Reviews & Ratings
- Notifications
- Analytics Dashboard

## Technology Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Django REST Framework

Database:
- PostgreSQL

## Benefits

- Fair prices for farmers
- Fresh products for consumers
- Transparent transactions
