# FarmConnect System Architecture

## Overview

FarmConnect is a Direct Farmer-to-Consumer Marketplace that enables farmers to sell products directly to consumers.

## Technology Stack

### Frontend

- React.js
- Tailwind CSS
- Axios
- React Router

### Backend

- Django
- Django REST Framework
- JWT Authentication

### Database

- PostgreSQL
